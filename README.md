# GeoIP Go Package

Wrapper package for the freegeoip.net api written in Go

## How to use
Via 'go get'
```
go get github.com/janritter/go-geo-ip/geoip
```

## Documentation
GoDoc - https://godoc.org/github.com/janritter/go-geo-ip/geoip

## How to test
Via 'gp test'
```
go test -v ./geoip 
```
